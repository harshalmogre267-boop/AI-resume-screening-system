# AI Resume Screening System - Product Requirements Document (PRD)

## 1. Project Overview

The AI Resume Screening System is a web-based application that automatically analyzes resumes, compares them with a job description, and ranks candidates based on skill matching and relevance.

## 2. Problem Statement

Recruiters spend significant time manually reviewing resumes. This process is slow, error-prone, and inefficient when handling hundreds of applications.

## 3. Objectives

* Automate resume screening.
* Reduce recruiter workload.
* Improve candidate shortlisting accuracy.
* Rank candidates based on job requirements.

## 4. Target Users

* HR Teams
* Recruiters
* Small and Medium Enterprises
* Placement Cells

## 5. Features

### Candidate Features

* Upload resume (PDF/DOCX)
* View matching score
* Receive skill-gap feedback

### Recruiter Features

* Upload job description
* Upload multiple resumes
* View ranked candidate list
* Download screening report

## 6. Functional Requirements

* Extract text from resumes.
* Extract skills from resumes.
* Compare resume skills with job description.
* Calculate matching score.
* Rank candidates.
* Display results through dashboard.

## 7. Non-Functional Requirements

* Response time less than 5 seconds.
* Secure file storage.
* User-friendly interface.
* Scalable architecture.

## 8. Success Metrics

* Resume processing accuracy > 90%.
* Candidate ranking accuracy > 85%.
* Processing time < 5 seconds per resume.

## 9. Expected Output

* Candidate Name
* Extracted Skills
* Matching Percentage
* Candidate Rank
* Missing Skills
